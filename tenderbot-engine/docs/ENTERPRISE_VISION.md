# Enterprise-тариф — видение «AI-помощник владельца бизнеса»

Дорожная карта Enterprise-уровня: AI, который проводит полный цикл анализа
лота от ТЗ до финплана. Зафиксировано на основе разговора с заказчиком
(ТОО Turanix), 2026-05-24.

## Конечная цель

Зашёл в лот → получил **полную картину**:
- что закупают (структурированно из ТЗ)
- где купить (поставщики/заводы)
- по какой цене + RFQ уже отправлены
- сколько везти и где (логистика)
- сколько на таможню (если импорт)
- какие риски (срок, валюта, надёжность)
- сколько отвлекать оборотки (предоплата, маржа, точка безубыточности)
- идти или нет → **GO/NO-GO**

## 10 модулей

1. **ТЗ→требования** — LLM из PDF/DOCX тянет структурированные требования
2. **Поиск товара** — каталоги поставщиков (Alibaba/Satu.kz/локальные)
3. **Сверка тех.характеристик** — match найденного товара с ТЗ
4. **Авто-RFQ** — LLM пишет запросы поставщикам, отправка
5. **Нормализация ответов** — таблица сравнения предложений
6. **Таможенная очистка** — HS-код → пошлина ЕАЭС → НДС → landed cost
7. **Логистика** — поиск перевозчиков, сроки, стоимость
8. **Риски** — дедлайн, валютный, репутация поставщика
9. **Финплан** — маржа, cashflow, отвлечение оборотки
10. **Дашборд лота** — всё на одном экране, GO/NO-GO

## Жёсткие реалии

### Экономика AI
Полный анализ лота через GPT/Claude — $2–5 в токенах.
1 клиент × 10 лотов/день × $3 = ~$900/мес.
**Текущий тариф Enterprise 100k ₸ (~$200) убыточен.**

**Решение:** либо поднять Enterprise до 300–500k ₸/мес, либо ввести
**pay-per-action** (5–10k ₸ за «глубокий анализ лота»), а подписка дешевле.

### Сложное — не AI, а данные
- ЕАЭС-тариф — открыт, ок
- База поставщиков РФ/КНР — нет готовой, нужны платные (ImportGenius/Alibaba API ~$1.5k/год)
- Перевозчики РК — нет API, нужны партнёрства
- Каталоги отраслевых заводов — фрагментированы

### Ответственность
AI ошибается. Везде нужны: confidence-скоры, human-in-the-loop,
аудит-трейл, disclaimer в договоре.

### Domain expertise
AI не знает отраслевых нюансов. Нужен эксперт в команде ИЛИ
накопленная история клиентских кейсов.

## Дорожная карта

### v1 — «AI-чек-лист» (3-4 недели)
- Парсер PDF/DOCX из приложений лота
- LLM-извлечение требований
- GO/NO-GO scorecard
- Финплан-калькулятор (cashflow, маржа, ТБУ)
- **Уже окупает 100k ₸/мес для активного поставщика**

### v2 — «AI-помощник по сорсингу» (2-3 месяца)
- RFQ-генератор (email + Whatsapp Business API)
- Email-инбокс с парсером ответов
- Простая таможня (HS-код + landed cost)
- Интеграция с Alibaba Lite / Satu.kz B2B

### v3 — «Risk + Logistics» (6+ месяцев)
- База поставщиков с историей (накопленная через клиентов)
- Risk scoring
- Партнёрства с перевозчиками (DAFI/КТЖ/Kazpost Cargo)
- Валютные риски

### v4 — «Auto-pilot» (12+ месяцев)
- Полная автоматизация с финальным human-approve
- Industry-specific агенты

## Технические зависимости

- LLM-провайдер: Claude Sonnet 4.x (баланс цена/качество)
  для основной работы, GPT-4o для сравнения
- PDF/DOCX парсинг: pdfplumber, python-docx, fitz
- Email: SMTP через Postmark/SendGrid + IMAP инбокс
- ЕАЭС-тариф: открытые данные с eec.eaeunion.org
- HS-код: GraphQL eaeunion + LLM-классификатор
- Поставщики (фаза v2+): Alibaba Open Platform, Satu.kz, ImportGenius

## Что НЕ делать в MVP

- Не покупать платные базы данных до первых $5k MRR
- Не интегрировать перевозчиков до v3 — слишком фрагментировано
- Не делать «auto-send» до v4 — клиент должен жать «отправить»
- Не браться за тех.характеристики без эксперта в команде
