# Тендерные площадки Казахстана — справочник

Карта площадок, на которых стоит мониторить лоты. Разбито на 3 группы по приоритету.
По каждой: URL, что там продают/покупают, что нужно для регистрации, есть ли API.

---

## Группа 1. Обязательно (по ТЗ)

### 1.1 Goszakup — государственные закупки РК ⭐ главный
- **URL:** https://goszakup.gov.kz
- **API:** ✅ GraphQL `https://ows.goszakup.gov.kz/v3/graphql` (v3)
- **Что внутри:** все закупки гос. органов и квазигоса (с порогом). Самый большой объём.
- **Регистрация:** через ЭЦП РК. Пройти регистрацию организации → получить роль "Администратор организации".
- **Токен API:** письмо в АО "Центр электронных финансов" → support@ecc.kz (см. README, раздел Goszakup).
  - Срок действия токена: 1 год.
  - Перевыпуск — в кабинете "Профиль участника → Выпуск токена".
- **Что нужно:**
  - `GOSZAKUP_TOKEN` (API)
  - `GOSZAKUP_LOGIN` + `GOSZAKUP_PASSWORD` (для автоподачи через браузер)

### 1.2 Самрук-Казына
- **URL:** https://www.samruk-zakup.kz
- **API:** ❌ публичного нет → парсинг через Playwright
- **Что внутри:** закупки нацхолдинга и дочерних (КМГ, КТЖ, КЕГОК, Казтелеком, Эйр Астана и др.) — крупнейший негосударственный заказчик.
- **Регистрация:** через ЭЦП РК.
- **Что нужно:** `SAMRUK_LOGIN` + `SAMRUK_PASSWORD`

### 1.3 ЕТК — Электронная торговая компания
- **URL:** https://etk.kz
- **API:** ❌ парсинг
- **Что внутри:** негосударственные закупки коммерческих компаний.
- **Регистрация:** через сайт, обычно нужна ЭЦП для подачи (но для просмотра — нет).
- **Что нужно:** `ETK_LOGIN` + `ETK_PASSWORD`

### 1.4 ЭТП
- **URL:** https://etp.kz
- **API:** ❌ парсинг
- **Что внутри:** коммерческие тендеры разных отраслей.
- **Регистрация:** через сайт + ЭЦП.
- **Что нужно:** `ETP_LOGIN` + `ETP_PASSWORD`

### 1.5 БТК — Биржевая торговая компания
- **URL:** https://btk.kz
- **API:** ❌ парсинг
- **Что внутри:** товарно-сырьевые торги.
- **Регистрация:** на сайте.
- **Что нужно:** `BTK_LOGIN` + `BTK_PASSWORD`

### 1.6 Казатомпром
- **URL:** https://zakup.kazatomprom.kz
- **API:** ❌ парсинг
- **Что внутри:** закупки атомной промышленности (НАК "Казатомпром" и дочки).
- **Регистрация:** ЭЦП обязательна.
- **Что нужно:** `KAZATOMPROM_LOGIN` + `KAZATOMPROM_PASSWORD`

## Группа 2. Дополнительно (расширить охват)

### 2.1 Атамекен — закупки субъектов квазигоссектора
- **URL:** https://goszakup.atameken.kz (или https://atameken.kz → раздел "Закупки")
- **Что внутри:** закупки субъектов, не подпадающих под Госзакуп (по 76 закону).
- **Регистрация:** ЭЦП РК.

### 2.2 КТЖ — Казахстан темир жолы (ЖД)
- **URL:** через samruk-zakup.kz (КТЖ — дочка Самрука)
- **Особенность:** объёмные закупки в логистике, строительстве, IT-инфраструктуре.

### 2.3 КазМунайГаз (КМГ)
- **URL:** через samruk-zakup.kz + собственный портал https://kmgep.kz
- **Что внутри:** нефтегаз, буровые услуги, оборудование.

### 2.4 KEGOC — национальная энергосеть
- **URL:** через samruk-zakup.kz; собственного отдельного нет.

### 2.5 Mitwork — частная маркетплейс-площадка
- **URL:** https://mitwork.kz, https://eep.mitwork.kz
- **Что внутри:** работа/услуги для МСБ. По духу ближе к фрилансу, но есть и тендеры.
- **Регистрация:** проще (email + телефон), без ЭЦП.

### 2.6 Платные агрегаторы
- **zakup.kz** — агрегатор объявлений со всех площадок.
- **tenders.kz / alemtender.kz** — мониторинг + аналитика.
- **Когда нужно:** если своих парсеров не хватит — агрегаторы дают единый API за деньги.

- **API:** есть REST + XML-выгрузки.
- **Когда нужно:** экспорт в РФ.

## Какие площадки реально стоят регистрации именно вам

Зависит от того, **что продаёт ваша компания**:

| Что вы продаёте | Какие площадки в приоритете |
|---|---|
| Услуги/работы для гос. органов | Goszakup + Атамекен |
| Оборудование, IT, строительство | Goszakup + Самрук-Казына + ЕТК |
| Нефтегаз, бурение | Самрук-Казына + КМГ (kmgep.kz) + Казатомпром |
| Транспорт, логистика, шпалы | Самрук-Казына (КТЖ внутри) |
| Сырьё, металлы, химия | БТК + ЭТП |
| Услуги для МСБ | Mitwork + Goszakup |

Регаться **на все** имеет смысл только если у вас широкий профиль. Лучше — на 2–3 самых релевантных.
