# Шаблон сбора реквизитов — заполнить и прислать обратно

Заполните то, что есть. Пустые поля можно оставить — система запустится частично.
Когда пришлёте — я впишу всё в `.env`, перезапущу сервер, и мониторинг начнёт реально работать.

## 11. Настройки мониторинга — фильтры лотов

```
MIN_LOT_PRICE = 1000000                   # минимальная цена лота (тенге)
MAX_LOT_PRICE = 500000000                 # максимальная цена
KEYWORDS      = строительство, IT, поставка    # ключевые слова в названии
REGIONS       = Алматы, Астана                  # ваши регионы работы
SCAN_INTERVAL = 30                         # пауза между сканами (секунды)
```
